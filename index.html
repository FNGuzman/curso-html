Demostración de Etiquetas HTML Principales

Esta página contiene ejemplos de las etiquetas más comunes utilizadas en HTML, junto con una breve explicación de cada una.

<!DOCTYPE>

La declaración <!DOCTYPE> define el tipo de documento y asegura que el navegador renderice la página correctamente. En HTML5 se usa simplemente <!DOCTYPE html>.

<html>

La etiqueta <html> es el contenedor principal de todo el contenido HTML. El atributo lang especifica el idioma del contenido.

<head>

La etiqueta <head> contiene metadatos sobre el documento, como el conjunto de caracteres (<meta charset="UTF-8">), el título (<title>), y otros enlaces importantes como hojas de estilo o scripts.

<title>

La etiqueta <title> define el título de la página, que aparece en la pestaña del navegador.

<meta>

La etiqueta <meta> proporciona metadatos, como la codificación de caracteres (<meta charset="UTF-8">) y descripciones para motores de búsqueda (<meta name="description" content="...">).

<body>

La etiqueta <body> contiene todo el contenido visible en la página web, como texto, imágenes, enlaces, etc.

<h1> a <h6>

Las etiquetas de encabezado <h1> a <h6> se utilizan para definir títulos y subtítulos, siendo <h1> el nivel más alto y <h6> el más bajo.

<p>

La etiqueta <p> se usa para definir párrafos de texto.

<a>

La etiqueta <a> define un enlace (hipervínculo). El atributo href especifica la URL a la que apunta el enlace. Ejemplo: <a href="https://www.example.com">Visitar Ejemplo</a>.

<img>

La etiqueta <img> se utiliza para insertar imágenes en la página. El atributo src define la ruta de la imagen y alt proporciona texto alternativo. Ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen">.

<ul>, <ol> y <li>

Las etiquetas <ul> (lista desordenada) y <ol> (lista ordenada) se utilizan para crear listas. Los elementos de la lista se definen con <li>.

<div> y <span>

Las etiquetas <div> y <span> son contenedores genéricos. <div> es un bloque que ocupa todo el ancho disponible, mientras que <span> es un contenedor en línea que solo ocupa el espacio de su contenido.

<footer>

La etiqueta <footer> se utiliza para definir el pie de página de un documento o sección, generalmente con información sobre el autor o enlaces adicionales.
